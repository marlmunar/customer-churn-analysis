## Customer Churn Analysis (SQL + Power BI)
# Overview
This project analyzes customer churn behavior using an e-commerce dataset. It combines SQL-based exploratory data analysis (EDA) with Power BI modeling and visualization to uncover churn drivers and provide actionable insights.

<img width="1352" height="766" alt="image" src="https://github.com/user-attachments/assets/4c1132ac-2588-44c0-ad6d-cf0cd102c918" />

# Objectives
- Understand customer churn patterns
- Segment customers based on value
- Identify key drivers of churn
- Provide data-driven recommendations

# Workflow
1. Data Exploration (SQL)
  - Loaded dataset into a local database
  - Performed EDA to validate data quality
  - Checked for inconsistencies and anomalies
2. Data Cleaning
  - Standardized column formats
  - Fixed data inconsistencies
  - Prepared analysis-ready dataset
3. Data Modeling (Power BI)
  - Built relationships between tables
  - Created calculated columns and measures
  - Structured data for efficient reporting
4. Visualization
  - Designed interactive dashboards
  - Highlighted churn trends and patterns

# Key Features
- Customer segmentation by revenue value
- Churn rate analysis
- Trend analysis based on customer tenure
- Identification of churn drivers
- Business recommendations based on insights

# Tools & Technologies
- SQL (for EDA and data preparation)
- Power BI
- DAX
- PowerQuery

# Skills Demonstrated
- SQL Data Exploration
- Data Cleaning
- Data Modeling
- Customer Segmentation
- Business Insight Generation
- Dashboard Design

Simulated e-commerce churn dataset from [source](https://mavenanalytics.io/data-playground/telecom-customer-churn)

- Explore churn trends and customer segments
